## Installation

### Prerequisites
- Node.js installed
- Git installed

1. Clone the repository:
    ```bash
    git clone https://github.com/Debasish121/techystar.git
    ```

2. Navigate to the project directory:
    ```bash
    cd techystar
    ```

3. Install dependencies:
    ```bash
    npm install
    ```
