## Installation

### Prerequisites
- Node.js installed
- Git installed

1. Clone the repository:
    ```bash
    git clone https://github.com/Debasish121/video-hub.git
    ```

2. Navigate to the project directory:
    ```bash
    cd video-hub
    ```

3. Install dependencies:
    ```bash
    npm install
    ```

## Usage
To start the development server, use the following command:
